#Quiz Project

#Installation:
To run this project locally , follow these steps:
1. git clone https://github.com/FariaAfrose/Quiz-project.git
2.cd Quiz-project
3.npm install
4.npm run dev
Open http://localhost:5173 to view it in the browser.
